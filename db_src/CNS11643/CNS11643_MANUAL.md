# CNS11643 全字庫 Manual

## 출처

- **발행**: 國家發展委員會 (대만 국가발전위원회)
- **공식 사이트**: https://www.cns11643.gov.tw
- **오픈데이터 포털**: https://data.gov.tw/dataset/5961
- **라이선스**: 정부 공개 데이터 (무료 사용, 출처 표기 권장)
- **수집일**: 2026-04-04
- **데이터 기준일**: 2025-03 (Properties), 2025-07 (Unicode mapping)

## 파일

### Properties/
- `CNS_phonetic.txt` — 주음부호 발음
- `CNS_pinyin_1.txt`, `CNS_pinyin_2.txt` — 한어병음
- `CNS_stroke.txt` — 총 획수
- `CNS_radical.txt` — 부수 번호
- `CNS_cangjie.txt` — 창힐 입력코드
- `CNS_component.txt` — 자형 구성 성분
- `CNS_strokes_sequence.txt` — 획순 시퀀스
- `CNS_source.txt` — 한자 출처 정보

### MapingTables/Unicode/
- `CNS2UNICODE_Unicode BMP.txt` — CNS → Unicode BMP 매핑
- `CNS2UNICODE_Unicode 2.txt`, `3.txt`, `15.txt` — 확장 평면 매핑
